## ❓ Frequently Asked Questions

**Q: Why are agents disabled in v3.2?**  
A: To ensure maximum stability and performance while we optimize the agent system for v3.3.

**Q: Will I need to pay again for v3.3?**  
A: No. v3.3 is a free update for all Genesis key holders.

**Q: Can I still use TitanU OS without agents?**  
A: Yes! All core features work perfectly in v3.2. You can chat, manage files, use memory, and scrape websites.

**Q: When exactly will v3.3 be released?**  
A: Q1 2026. We'll announce the exact date as we get closer to release.

**Q: What if I don't have a Genesis Key?**  
A: TitanU OS requires a Genesis Key for access. Contact the team for information on obtaining one.
